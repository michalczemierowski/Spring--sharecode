# Sharecode
Java Version: **2019.3.0f1**

Spring Boot Version: **2.3.2**

[Ace Editor](https://github.com/ajaxorg/ace) Version: **1.4.12**


Features:
- Authentication with OAuth2 (GitHub/Google)
- Room content is updated realtime using SSE
- Creating own room
- Adding/Removing acces to room for other users
